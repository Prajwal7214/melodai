# 🎵 MelodAI
### AI-Based Music Composition + Recommendation System

## Quick Start

### Backend
```bash
cd backend
venv\Scripts\activate
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Tech Stack
- Backend: Django + MongoDB Atlas
- Frontend: React + Tailwind + Framer Motion
- Music: iTunes API
- AI: MelodAI Generator v1.0
